🖤 CULT SOCIETY
어릴 때부터 일본 빈티지를 좋아해 해외 직구와 세컨핸드 구매를 자주 했습니다. 그런데 국내에서 매물을 찾으려면 후루츠패밀리, 번개장터, 중고나라 등 여러 플랫폼을 동시에 뒤져야 했고, 유행이 붙은 아카이브 피스는 시세가 발매가의 몇 배를 웃도는데도 이를 비교할 수 있는 서비스가 없었습니다. CULT SOCIETY는 그 불편함에서 출발했습니다. 아카이브 패션 피스의 시세 추적 + 중고 거래를 한 곳에서 — 후루츠패밀리의 감성과 KREAM의 호가 시스템을 결합한 플랫폼입니다.
"죽지 않는 옷들의 시장. 유행은 끝나도 가치는 남는다."


✨ 주요 기능

시세 추적 — 아이템별 발매가 대비 등락률, Chart.js 기반 거래가 추이 그래프 (3개월/6개월/1년), 52주 최고·최저 비교
거래 시스템 — 즉시 구매 / 입찰 제시 호가 방식, 현재 입찰 현황 실시간 표시
아이템 상세 페이지 — 카드 클릭 시 오버레이 팝업, 상품별 고유 시세 차트 + 거래 이력 테이블
브랜드 아카이브 — 희소성·시세·거래량 종합 랭킹 1–20위, 브랜드 클릭 시 히스토리 + 인기상품 TOP 3 팝업
MD 큐레이션 — 매주 MD 선정 아카이브 피스 에디토리얼
유저 옷장 프로필 — 보유 피스, 거래 이력, 위시리스트 탭, 취향 태그
